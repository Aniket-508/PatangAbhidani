## Patang Abhidani - A comprehensive guide for lepidopterists and butterfly-enthusiasts.

### Brief Summary :
The primary motivation for the project was that Skipper butterflies arenâ€™t well documented in India. Due to rapid urbanization, butterfly counts have been receding and it is important to track the migration patterns of these butterflies to prevent them from going extinct. The system built would be able to accomplish this by gathering user submitted data and verifying its validity. This would enable new researchers to get information about skippers without having to go through the plethora of websites online. It would also work as a good platform for users to share photographs clicked by them and hence increase the total data available online.
This repository contains code for full-stack website which was built as a part of our Third Year Engineering project for a local NGO named 'Green Works Trust' and successfully submitted to them after the given period.
